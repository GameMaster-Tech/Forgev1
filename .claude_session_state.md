## 1. Overall goal

Build Forge — an "Epistemic Compiler" research workspace — with three
flagship features:

- **Sync** — cross-document constraint engine (Budget vs Hiring paradoxes, market-anchored auto-patches).
- **Pulse** — temporal entropy / truth-decay engine (reality-diff against market mocks, document refactors).
- **Lattice** — recursive task decomposition (`decomposeTask` → atomic subtasks bound to assertions + draftOutcomes).

Plus a **Calendar + Tempo** layer: bidirectional Google Calendar sync, an
AI-native scheduler (priority engine, conflict detection, focus-block
packer, goal-block placement), habits with streak math, multi-tab
realtime via SSE, sharing/permissions, and cron-backed maintenance.

Stack: Next.js 16 (webpack in this worktree, Turbopack in main repo) ·
React 19 · TipTap · Tailwind 4 · Firebase (client + admin) · Anthropic
SDK · Exa.

### TASK 1 — Fix the typecheck-blocking `commitChunked` helper (P0, blocks everything else)

**File:** `src/app/api/integrations/google/sync/route.ts`

The audit pass started refactoring the Firestore batch logic to chunk
into ≤400-op groups (Firestore caps at 500). The route currently calls
**`commitChunked(fs, eventsCol.path, localBatchedOps)`** and
**`commitChunked(fs, snapCol.path, snapOps.map(...))`** — but the
`commitChunked` helper has **not been written yet**. Typecheck will
fail until it lands.

Drop this helper at the bottom of the file:

```ts
async function commitChunked(
  fs: FirebaseFirestore.Firestore,
  collectionPath: string,
  ops: Array<{ op: "set" | "delete"; path: string; data?: unknown; merge?: boolean }>,
  chunkSize = 400,
): Promise<void> {
  const col = fs.collection(collectionPath);
  for (let i = 0; i < ops.length; i += chunkSize) {
    const slice = ops.slice(i, i + chunkSize);
    const batch = fs.batch();
    for (const op of slice) {
      const ref = col.doc(op.path);
      if (op.op === "set") {
        batch.set(ref, op.data as FirebaseFirestore.DocumentData, op.merge ? { merge: true } : undefined);
      } else {
        batch.delete(ref);
      }
    }
    await batch.commit();
  }
}
```

commit per micro-fix or roll all of TASK 1 into one commit.
